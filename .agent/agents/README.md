## üîÑ How Agents Work

### Individual Use
Each agent can be referenced individually for specialized expertise in their domain.

### Orchestrated Use (Recommended)
Agents are most powerful when orchestrated together through workflows:

| Workflow | Agents Used | Execution |
|----------|-------------|-----------|
| `/feature-flutter` | Grand Architect ‚Üí Security ‚Üí Performance ‚Üí Test Generator | Sequential + Parallel QA |
| `/review-flutter` | All 7 agents | **Parallel Execution** (2-3x faster) |
| `/stitch-flutter` | Stitch Converter ‚Üí A11y ‚Üí Design Guardian | **Design-to-Code** |
| `/test-flutter` | Test Generator | Focused test generation |

> **‚ö° Parallel Execution:** Review workflows execute multiple agents simultaneously for 2-3x faster reviews.
